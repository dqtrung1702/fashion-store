Quyết định đã thể hiện trong code

  • Admin thao tác ảnh qua API, không dùng credential MinIO ở trình duyệt.
  • Ảnh sản phẩm dùng prefix products/ trong MinIO.
  • Xóa ảnh đang được sản phẩm tham chiếu bị chặn ở giao diện và API.
  • Một sản phẩm có thể thuộc nhiều collection qua collectionSlugs.
  • Redis là cache mềm cho public products/collections/categories/content; MongoDB vẫn là nguồn dữ liệu gốc.
  • Nginx production serve frontend static và proxy API/assets, không public MongoDB/Redis/backend.
  • Order item dùng snapshot để giữ lịch sử mua hàng ổn định.

Cần quyết định trước production

  • HTTPS trực tiếp bằng Nginx/Certbot hay đặt sau Cloudflare/CDN.
  • Bucket public/private, CDN hoặc signed URL cho media.
  • Phân quyền admin chi tiết hơn ngoài is_admin.
  • Nhật ký thao tác admin và chiến lược backup MongoDB/MinIO.
  • Cổng thanh toán, hoàn tiền, đối soát và webhook.
