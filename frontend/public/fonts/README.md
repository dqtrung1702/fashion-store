The active storefront font pairing is kept local so the site does not depend on external font services at runtime:

- `UVNMuaThu.ttf`
- `Nunito-400.ttf`
- `Nunito-500.ttf`
- `Nunito-600.ttf`
- `Nunito-700.ttf`

`UVN Mua Thu` is used for logo, display headings, hero titles, campaign titles, and section titles. `Nunito` is used for body copy, navigation, buttons, forms, and admin screens.

`Lora-400.ttf`, `Lora-500.ttf`, `Lora-600.ttf`, `Lora-700.ttf`, `VLSelfie-Regular.otf`, `NVNAzkia.ttf`, and `1FTVBillionDreams.ttf` are kept here as previous local font options but are no longer referenced by the active stylesheet.

The active Nunito files were downloaded from Google Fonts. The previous `VLSelfie-Regular.otf` was imported from Font2s; Font2s lists it as "Free for Personal Use", so it should stay inactive unless the license is confirmed for production use.
